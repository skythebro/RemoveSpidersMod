# Remove Spiders Mod for Sons of the Forest

This mod removes spiders and spider webs from Sons of the Forest.

## Features

- **No Pickup Spiders**: Prevents spiders from spawning when you pick up rocks, logs, or other items.
- **No Spider Webs**: Completely removes spider web objects from the game world.
- **No Web Spawners**: Prevents spiders from spawning from webs.

## Installation

### Prerequisites

- [RedLoader](https://github.com/ToniMacaroni/RedLoader) installed.
- [Red Manager](https://github.com/ToniMacaroni/RedManager) (recommended).

### Installing with Red Manager

1. Download the latest `RemoveSpidersMod.zip` from the [Releases](https://github.com/skythebro/RemoveSpidersMod/releases) page.
2. Open **Red Manager**.
3. Drag and drop the downloaded `.zip` file into the Red Manager window.
   - Alternatively, use the **Import** button in Red Manager.
4. Ensure the mod is enabled in the list.
5. Launch the game via Red Manager.

### Manual Installation

1. Download the mod files.
2. Navigate to your Sons of the Forest game directory.
3. Go to the `Mods` folder inside the RedLoader installation (usually `Sons Of The Forest/Mods`).
4. Extract the `RemoveSpidersMod.dll` and `manifest.json` into a new folder named `RemoveSpidersMod`.
   - Structure should look like: `Sons Of The Forest/Mods/RemoveSpidersMod/RemoveSpidersMod.dll`

## Building from Source

1. Clone the repository.
2. Open `RemoveSpidersMod.sln` in your IDE (JetBrains Rider or Visual Studio).
3. Build the project (Release mode recommended).
4. The output zip will be in `ReleaseBuild/RemoveSpidersMod.zip`.
